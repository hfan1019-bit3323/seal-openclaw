# A股板块轮动规律

## 核心规律：A股不是涨指数，是涨板块

A股的赚钱效应主要体现在板块轮动上，而非指数单边上涨。
典型节奏：一个主线持续1-3周 → 过热后资金撤出 → 切换到新方向。

## 板块轮动的常见路径

### 事件驱动型轮动
```
突发事件（中东冲突/政策出台）
  → 第一天：直接受益板块暴涨（油气/军工）
  → 第2-3天：二级受益板块跟涨（化工/煤炭/航运）
  → 第3-5天：替代逻辑板块启动（新能源/储能/绿电）
  → 第5-10天：产业链逻辑板块成为主线（锂电材料/电解液）
  → 第10天+：前期暴涨板块开始调整，资金切向新方向
```

### 产业周期型轮动
```
涨价/业绩拐点出现
  → 龙头先行（宁德时代/赣锋锂业）
  → 中游跟进（天赐材料/恩捷股份）
  → 上游扩散（融捷股份/盛新锂能）
  → 边缘品种补涨（小市值概念股）
  → 全面过热后调整
```

## "跷跷板"效应

A股经常出现明显的风格跷跷板：
- **通胀交易 vs 成长交易**：油价涨→化工/煤炭涨→科技跌；油价跌→反转
- **大盘 vs 小盘**：权重护盘时小盘跌，权重休息时小盘补涨
- **红利 vs 成长**：避险时买银行/电力，进攻时买科技/锂电

## 判断板块是否过热的信号
1. 连板龙头出现"天地板"或大幅回落
2. 板块内涨停家数从高峰开始减少
3. 龙虎榜显示机构开始大规模净卖出
4. 板块换手率飙升到极端值（>40%）
5. 公司集中发布"股价异常波动"公告

## 判断新主线正在形成的信号
1. 连续3天以上有资金净流入，且流入金额递增
2. 产业链上中下游联动（不是单只个股涨停）
3. 在大盘下跌日能逆势走强
4. 龙虎榜显示机构+北向共同净买入
5. 基本面有独立于市场情绪的催化（涨价/业绩/政策）

## 中东冲突期间（2026年3月）的特殊轮动节奏
- 第1周：恐慌杀跌→ETF抄底→油气/煤炭暴涨
- 第2周：化工涨价传导→电力板块爆发→锂电材料启动
- 第3周：电力龙头断板→资金切向锂电和创新药
- 第4周：停火消息→油气/化工暴跌→科技/成长修复
- 规律：A股对地缘冲击的"脱敏"速度约2-3周
