# USER.md - 目标用户

## 直接用户

- **团队规模**：3-10 人，私募/工作室/独立研究团队
- **市场**：A 股为主，偶尔关注港股/美股联动
- **交易工具**：同花顺、东方财富等（擎鲸不替代这些）
- **技术水平**：会用 Excel 和行情软件，不写代码
- **信息来源**：研报、新闻、龙虎榜、资金流、板块轮动
- **痛点**：信息过载、复盘耗时、团队信息不同步

## 用户不是谁

- 不是日内高频交易者（不需要毫秒级行情）
- 不是纯量化团队（不需要自己写策略代码）
- 不是个人散户（不做"一键跟单"或"荐股"）
- 不是 IT 运维人员（不需要看日志、管服务器）

## 使用场景

### 每日必用

| 场景 | 用户期望 | 擎鲸输出 |
|------|---------|---------|
| 盘前看一眼 | 隔夜有什么影响今天的？ | 晨间简报：美股/期货/政策 → 对 A 股影响 |
| 盘中异动 | 关注的板块/标的有动静？ | 事件驱动推送：资金异动、涨停板、重大公告 |
| 收盘复盘 | 今天发生了什么，明天关注什么？ | 日报：概览 → 板块 → 资金 → 龙虎榜 → 明日策略 |

### 高频使用

| 场景 | 用户期望 | 擎鲸输出 |
|------|---------|---------|
| 对话提问 | "锂电今天为什么跌？" | 板块分析 + 资金流向 + 消息面归因 |
| 研报消化 | "帮我看看这份研报" | 核心观点 + 关键数据 + 与持仓关联性 |
| 信号验证 | "XX 龙虎榜怎么样？" | 龙虎榜明细 + 机构动向 + 历史对比 |
| 策略回顾 | "上周判断对了几个？" | 周复盘 + 命中率 + 偏差分析 |

## 交互期望

**想要的：**
- 打开页面就能看到有用信息（不是空白等我提问）
- 分析结果一目了然（表格、图表内联在消息流里）
- 推送及时准确（异动提醒不能迟到 10 分钟）

**不想要的：**
- 切换多个页面才能看完一个分析
- 被无关信息轰炸（非交易日收到盘中提醒）
- 看一堆技术术语不知道该怎么做
